# 🤖 Robot Limpiador Virtual

Proyecto de Inteligencia Artificial que simula un robot autónomo capaz de limpiar un área, evitar obstáculos y optimizar rutas de limpieza.

## 🚀 Tecnologías
- Python 3.9+
- Programación orientada a objetos
- Algoritmos de planificación de rutas

## 🧩 Funcionalidades
- Diseña el comportamiento del robot
- Simula navegación a puntos específicos
- Limpia áreas de manera eficiente evitando repetir espacios
- Optimiza rutas considerando batería y capacidad de basura

## ▶️ Ejecutar proyecto

1. Clonar repositorio:
```bash
git clone <URL-del-repositorio>
cd robot_virtual
```
---

## ▶️ Ejecución

```bash
python robot_limpiador.py

```
